CSS3动画实现一个盒子在2S内反复上下移动

2018-09-17   CSS 

前段时间遇到个笔试题，大概就是要求用CSS实现一个盒子在2S内匀速反复上下移动，兼容主流浏览器。

使用CSS3中的@keyframes规则和animation属性可以实现动画效果,代码如下:

<!DOCTYPE html>
<html>
<head>
<style> 
div
{
    width:100px;
    height:100px;
    background:red;
    position:relative;
    animation:mymove 2s infinite alternate linear;/*作用于mymove,2s完成一次动画,动画次数无限次,一次动画播放完后反向播放,动画速度匀速*/
    -moz-animation:mymove 2s infinite alternate linear; /* Firefox */
    -webkit-animation:mymove 2s infinite alternate linear; /* Safari and Chrome */
    -o-animation:mymove 2s infinite alternate linear; /* Opera */
}

@keyframes mymove
{
    0%   {top:0px;}
    100% {top:200px;}
}

@-moz-keyframes mymove /* Firefox */
{
    0%   {top:0px;}
    100% {top:200px;}
}

@-webkit-keyframes mymove /* Safari and Chrome */
{
    0%   {top:0px;}
    100% {top:200px;}
}

@-o-keyframes mymove /* Opera */
{
    0%   {top:0px;}
    100% {top:200px;}
}
</style>
</head>
<body>

<p><b>注释：</b>本例在 Internet Explorer 中无效。</p>

<div></div>

</body>
</html>

此处不知为何，firefox浏览器(windows 60.0 64位)中居然更改-moz-animation属性无效，更改-webkit-animation属性才有效，简直匪夷所思……

这里 animation:mymove 2s infinite alternate;也可以写成：

animation-name:mymove;  /*规定 @keyframes 动画的名称*/
animation-duration:2s;  /*规定动画完成一个周期所花费的秒或毫秒。默认是 0*/
animation-direction:alternate;  /*     规定动画是否在下一周期逆向地播放。默认是 "normal"*/
animation-timing-function:linear /*规定动画的速度曲线。默认为ease,linear为匀速*/

@keyframes规则具体用法与更多动画属性请参考：

http://www.w3school.com.cn/css3/css3_animation.asp
